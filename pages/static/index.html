Private Share
Private Share is a simple file sharing web application which priorities privacy.
This is accomplished by forcing the use of a password allowing files to be
safely encrypted while they are stored on the server.

Why use Private Share?
  1. Files only stay on the internet for an hour.
  2. Files are encrypted with openssl while stored on the server.
  3. An Email will be sent to you when the file has been claimed.
