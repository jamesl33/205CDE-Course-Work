Private Share

Private Share is a simple file sharing web application which
priorities privacy. This is accomplished by forcing the use of
a password allowing files to be safely encrypted while they
are stored on the server.

Why use Private Share?

  1. Files only stay on the internet for an hour.
  2. Files are encrypted with openssl while stored on the server.
  3. An Email will be sent to you when the file has been claimed.

Upload

To upload a file simply navigate to the upload page.

  1. Drag and drop a file into the denoted area or select a file.
  2. Optionally enter an email your email (This will be used to notify you that your download link has been claimed).
  3. Enter a password (This will be used to encrypt the file).
  4. Press the submit button.

  • The stored file will be automatically purged from the server after and hour.
  • The file will only be available for one download.

Download

To download a file you will need a 'download' link. This will have been given to you by the person who uploaded the file.

  1. Enter the download URL into the search bar on your browser.
  2. Enter the password which was chosen when the file was uploaded.
  3. You should then get a download which is the decrypted file.

  • Once the file has been downloaded
  • Bear in mind that if you cancel the download the file will still be removed from the server.
